# Note Management App

A simple note-taking app built with React, TypeScript, and Vite. Features include:

- Add, edit, and delete notes.
- Light and dark theme support.
- Search functionality to filter notes.

## Technologies Used
- React
- TypeScript
- Vite
- CSS Modules
